3.0的新功能：

Texture

  • sRGB textures and framebuffers
  • 2d texture arrays—一个texture对象可以存储一组2d贴图，可以用于贴图动画。
  • 3d texture
  • Depth texture and shadow comparison—深度贴图标准支持，可用于shadow map。
  • Seamless cubemaps
  • Floating-point textures—浮点贴图，贴图像素每个R、G、B、A可存储16、32位浮点数，可用于HDR。
  • ETC2/EAC texture compression
  • Integer textures
  • Additional texture formats
  • Non-power-of-2 textures (NPOT)
  • Texture level of detail (LOD) features
  • Texture swizzles
  • Immutable textures
  • Increased minimum sizes

Shaders

  • Program binaries—可存储离线program对象，使用时直接加载，以节省shader编译、链接时间。
  • Mandatory online compiler
  • Non-square matrices
  • Full integer support
  • Centroid sampling
  • Flat/smooth interpolators
  • Uniform blocks
  • Layout qualifiers
  • Instance and vertex ID
  • Fragment depth
  • New built-in functions
  • Relaxed limitations

Geometry

  • Transform feedback—可以把顶点shader的输出存入buffer。
  • Boolean occlusion queries
  • Instanced rendering—高效渲染大量同类物体，如植被。
  • Primitive restart
  • New vertex formats

Buffer Objects

  • Uniform buffer objects
  • Vertex array objects
  • Sampler objects—贴图采样器与贴图分离，可被多个贴图对象共享。
  • Sync objects
  • Pixel buffer objects
  • Buffer subrange mapping
  • Buffer object to buffer object copies

FrameBuffer

  • Multiple render targets (MRTs)—一次渲染到多个目标，可用于延迟渲染。
  • Multisample renderbuffers
  • Framebuffer invalidation hints
  • New blend equations
