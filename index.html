Viry3D

C++ cross platform 3D game engine.
Based on Direct3D 11 for Universal Windows Platform,
and OpenGL ES 2.0/3.0 for Android and iOS.
With Unity3D like API programming style.

FEATURES

Programming language

  • C++ 11 for engine core

Platform & 3D API support

  • Direct3D 11, OpenGL ES (2.0, 3.0) support
  • Universal Windows Platform, Mac OS X, Android, iOS support

Material / Shader support

  • Supports vertex and fragment shaders, both HLSL and GLSL written in one shader file
  • Multiple pass effects
  • Load textures from png, jpeg

Mesh

  • Export meshes with Unity3D
  • Static geometry batching

Animation

  • Export animations with Unity3D
  • Skinned mesh support
    • blending of multiple animations with variable weights
    • variable/multiple bone weight skinning
    • hardware-accelerated skinning
  • Bezier curves based AnimationCurve

Scene Features

  • GameObject-Component programming model
  • Transform based hierarchical scene graph, nodes allow objects to be attached to each other and follow each others movements, articulated structures etc

Special Effects

  • HDR rendering pipeline support
  • Deferred shading pipeline support
  • Directional, spot, point lights
  • Shadow mapping, support CSM
  • Lightmap support
  • Flexible camera component
  • ImageEffect system, allowing for full-screen postprocessing effects to be combined easily
  • Bloom, Global Fog, Highlighting effect
  • Particle Systems, automatic use of particle pooling for maximum performance
  • Support for Skybox
  • Transparent objects automatically managed
  • Terrain rendering

Physics

  • Based on Bullet Physics Engine
  • BoxCollider
  • MeshCollider
  • TerrainCollider

UI Features

  • Sprite renderer for drawing image
  • Text renderer for drawing label using font

Audio

  • OpenAL based audio engine for all platforms
  • Support for wav, mp3 (streaming decode)

Input

  • Mouse and Touches

Misc features

  • File IO
  • UTF8, UTF32 string encoding
  • Mathf library

Cross Platform Support

Viry3D supports Universal Windows Platform, Mac OS X, Android, iOS.
